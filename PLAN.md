# خطة موقع شركة "خليك مكانك" - توصيل طلبات

## معلومات الشركة
- **الاسم:** خليك مكانك
- **نوع الخدمة:** توصيل متعدد (طعام، طرود، منتجات)
- **المنطقة:** محافظة الفيوم
- **الألوان:** أصفر وأبيض

---

## المرحلة الأولى: هيكل الموقع

### 1. Landing Page (صفحة الهبوط)
- **Header:** شعار الشركة + اسم "خليك مكانك" + قائمة تنقل
- **Hero Section:** صورة رئيسية مع نص "خليك مكانك واحنا نوصلك" + زر "اطلب دلوقتي"
- **Services Section:** 3 بطاقات (توصيل طعام، توصيل طرود، توصيل منتجات)
- **How It Works:** 3 خطوات (اختار، ادفع، استلم)
- **About Us:** معلومات عن الشركة
- **Testimonials:** آراء العملاء
- **Contact:** رقم التليفون + الواتساب + العنوان
- **Footer:** روابط التواصل الاجتماعي + حقوق النشر

### 2. Admin Panel (لوحة التحكم)
- **Dashboard:** إحصائيات عامة (عدد الطلبات، الإيرادات، العملاء)
- **Orders Management:** عرض وإدارة جميع الطلبات
- **Customers Management:** إدارة بيانات العملاء
- **Delivery Drivers:** إدارة مناديب التوصيل
- **Settings:** إعدادات الموقع

---

## المرحلة الثانية: التقنيات

### Frontend (الواجهة الأمامية)
- **HTML5:** هيكل الصفحة
- **CSS3:** التصميم بالألوان (أصفر + أبيض)
- **JavaScript:** التفاعلية
- **Bootstrap 5:** التصميم المتجاوب

### Backend (لوحة التحكم)
- **Firebase Authentication:** تسجيل الدخول للadmin
- **Firebase Firestore:** قاعدة البيانات
- **Firebase Storage:** تخزين الصور

---

## المرحلة الرابعة: التصميم بالتفصيل

### ألوان الموقع الرسمية
- **اللون الأساسي:** `#FFD700` (ذهبي/أصفر)
- **اللون الثانوي:** `#FFFFFF` (أبيض)
- **لون النصوص:** `#333333` (رمادي غامق)
- **لون الخلفية:** `#FFF8E7` (أصفر فاتح جداً)

### الخطوط
- **العربية:** Cairo أو Tajawal
- **الإنجليزية:** Poppins
